# LiMon_Display

## Overview

Simple ESP8266 traffic monitor to work with (https://github.com/xba1k/wifi_limon)

## Setup

This project was created using VSCode/PlatformIO.

# Hardware

Hardware is a basic ESP8266 Dev Module with 0.96" OLED on pins D5/D6.
